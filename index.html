# web2.1 Phân tích thành phần URL

Công cụ phân tích thành phần URL

Dưới đây là kết quả phân tích chi tiết các thành phần cấu tạo nên một đường dẫn URL tiêu chuẩn.

URL mẫu:

https://example.com/products/list?page=2#top

Danh sách các thành phần phân tích:

Thành phần (Component) Giá trị tách bóc (Value) Ý nghĩa (Description)
Protocol (Giao thức) https Phương thức truyền tải dữ liệu an toàn giữa trình duyệt và server.
Domain (Tên miền) example.com Địa chỉ định danh của trang web trên mạng Internet.
Path (Đường dẫn) /products/list Vị trí hoặc tài nguyên cụ thể đang được truy cập trên server.
Query (Tham số truy vấn) page=2 Dữ liệu gửi kèm lên server (thường dùng để lọc, phân trang).
Fragment (Thẻ neo) #top Định vị đến một vùng/vị trí cụ thể bên trong trang HTML hiện tại.
